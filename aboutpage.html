About

This website is a personal project that shows my artwork in a portfolio-style format.

The purpose of creating it is to practise HTML and CSS while combining my coding and illustration skills.

Each character is designed with their own unique identity and simple background, drawn in a cute anime, family-friendly style.

Visitors can view character designs and explore how the layout and presentation evolve over time.

This project is created by a diploma student in ICT with an interest in coding and AI.

The project will continue by improving designs, more detailed Characters and stronger presentation.
